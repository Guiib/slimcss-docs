O Slim

Você deve usar o Slim se estiver embarcando em um projeto menor ou simplesmente não sinta que precisa de toda a utilidade de estruturas maiores. O Slim apenas define um bloco de elementos HTML padrão e inclui um grid, mas isso geralmente é mais do que suficiente para começar. Na verdade, este site é construído no Slim.

Tipografia

A tipografia está definida com o tamanho em rems, então os tamanhos de fontes e as relações espaciais podem ser dimensionadas de forma qualificada com base em uma propriedade <html> de tamanho de fonte única. Fora da caixa, o Slim nunca muda o tamanho da fonte do <html>, mas caso você precise de seu projeto todas as medias ainda são base 10, em bora assim, <h1> com 5.0rem o tamanho da fonte apenas sigifica 50px.

A tipografia de base usada é Raleway servido pelo Google, definido em 15rem (15px) com um line-height: 1.6 (24px). Outros tipos básicos como Links, Forte, ênfase e sublinhados estão obviamente incluídos.

Alertas

O Slim, assim como alguns frameworks mais conhecido possui classes de alertas, responsáveis por indicar cada ação feita, por exemplo, validação de formulários, veja abaixo alguns exemplos.

.alert .alert-flat-info
.alert .alert-flat-success
.alert .alert-flat-danger
.alert .alert-flat-warning

Onde, info, indica uma ação de informação, success indica uma ação de sucesso, danger indica uma ação de erro, e warning indica uma ação de atenção.

Botões

Os botões vêm em dois estilos básicos. O button-flat-default elemento padrão é simples, enquanto o .button-flat-info é vibrante e proeminente. Os estilos de botões são aplicados a vários elementos de formulário apropriados, mas também podem ser anexados arbitrariamente a links com uma classe .button .button-flat-*. Use a classe .block para usar a largura total da div pai.

Formulários

Os formulários são uma enorme dor, mas espero que esses estilos tornem um pouco mais fácil. Todas os inputs, selects e buttons são normalizados para um cross-browser de altura comum para que os inputs possam ser empilhados ou colocadas uma em relação à outra.

Classes de utilidade pública

O Slim possui uma série de pequenas classes de utilidade que atuam como ajudantes fáceis de usar. Às vezes, é melhor usar uma classe de utilidade do que criar uma nova classe inteira apenas para flutuar um elemento. 

							
/* Utility
–––––––––––––––––––––––––––––––––––––––––––––––––– */
/* Centraliza um elemento */
.block {
  display: block;
  width: 100%;
}

/* Esquerda, direita */
.left   { float: left; }
.right  { float: right; }

/* Alinhamento de textos */
.text-left      { text-align: left; }
.text-right     { text-align: right; }
.text-center    { text-align: center; }
.text-justify   { text-align: justify; }
.text-justify   { text-align: justify-all; }

/* Formatações de textos */
.text-lowercase   { text-transform: lowercase; }
.text-uppercase   { text-transform: uppercase; }
.text-capitalize  { text-transform: capitalize; }
